##  Project Description
An end-to-end data analytics project focused on analyzing e-commerce product data to uncover trends in pricing, discounts, ratings, and brand performance. The project combines Python-based Exploratory Data Analysis (EDA) with SQL-based data querying for deeper business insights.

---

##  Objectives
- Understand product pricing and discount strategies  
- Analyze product ratings across brands  
- Identify patterns in FK Advantage products  
- Perform data validation using SQL  
- Generate actionable insights for decision making  

---

## Technologies Used
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- MySQL & SQL Queries
- Jupyter Notebook
- Data Cleaning and Preprocessing

---

## Dataset Details
- Platform: Flipkart E-commerce  
- Total Records: **18,788**  
- Attributes: product name, price, discount, rating, brand, category, FK Advantage status

---

##  Key Features Implemented

### Python Analysis
- Data Cleaning and Transformation  
- Discount and Discount Percentage Calculation  
- Rating Distribution Analysis  
- Price vs Rating Visualization  
- Outlier Detection  

### SQL Analysis
- Database creation and data import  
- Aggregation and grouping queries  
- Brand-wise and rating-wise analysis  
- Price range segmentation  

---

##  How to Use
1. Open the Jupyter Notebook file  
2. Run all cells to reproduce EDA  
3. Import dataset into MySQL  
4. Execute provided SQL queries
